## 🔧 기술적 개선사항
### **통합 관리**
- 모든 HTML → `css/style.css` 참조
- 모든 HTML → `js/script.js` 참조
- 중복 코드 완전 제거

### **확장성**
- 새 슬라이드 추가 시 CSS/JS 링크만 추가
- 스타일 수정 시 css/style.css 하나만 편집
- 기능 수정 시 js/script.js 하나만 편집

## 🚀 사용법
1. **시작**: `index.html` 열기
2. **목차**: 우측 상단 "목차" 버튼 클릭
3. **네비게이션**: 좌우 화살표 또는 키보드 방향키
4. **풀스크린**: F11 키 (권장)
